To use the mailer associated with this program,
Start the web2py server.
open the command line and navigate to the web2py folder.

Enter python web2py.py -S mailer -M -N -R applications/amberdoctor_tasktracker2/private/mail_task.py

There is no need to copy the file out of the plugin into the private folder to as
there should already be a copy there.

Static copy in the plugin_mailer is meant for reference.
